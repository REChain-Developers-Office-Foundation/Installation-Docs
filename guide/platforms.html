# Platforms

# Create Platform

  • Title: The name of the platform.
  • Slug: A user-friendly URL identifier for the platform.
  • Icon: An icon to represent the platform.
  • Image: A visual representation of the platform.
  • Footer: Indicate if the platform should appear in the website's footer.
  • Navbar: Determine if the platform should be included in the website's navigation bar.
  • Right Column: Choose whether the platform should be visible in the right column.

# Translations

This section becomes active after adding the platform, allowing you to translate any platform you desire or delete the translations you've added. This section utilizes the Google Translate infrastructure. Excessive requests may temporarily disable this feature. If you encounter such an issue, please retry after some time.

# SEO Settings

  • Custom Meta Title: Create a custom title for SEO metadata related to the platform.
  • Custom Meta Description: Specify a custom description for SEO metadata.
  • Custom H1 Title: Define a custom H1 title for SEO optimization.

This section is designed for customizing specific SEO elements, including meta title, meta description, and H1 tags. You can use the following supported shortcodes to dynamically generate these elements:

  • %platform_title% Inserts the title of the platform.
  • %site_title% Represents the title of the website.
  • %sep% Adds a separator between elements.
  • %year% Provides the current year.
  • %month% Indicates the current month in numeric format.
  • %day% Shows the current day of the month.
  • %month_text% Presents the name of the current month.
  • %day_text% Displays the name of the current day of the week.

# Other Details

# Sorting Platforms

To reorder platforms, navigate to the platforms page and click the "Sort Items" button. This will open a list of platforms where you can change their order by simply clicking and dragging them up or down using your mouse.

# Difference of Icon vs Image

You can use both icons and images for platforms. You can select an icon from hundreds of Font Awesome icons or upload a custom image for a platform. If an image is used, the icon will not be displayed. The chosen icon or image will be visible in both the site's main menu (navbar) and the platform section on the right-hand side.

# Hide/Delete Icons or Images

If you don't want to use an icon, please select None at the beginning of the icon list. To delete the uploaded image, click on the red x button on the image. In this case, no icon or image will be displayed next to the platform title.
