# Topics

# Create Topic

  • Title: The name of the topic.
  • Slug: A user-friendly URL identifier for the topic.
  • Description: A brief summary of the topic.
  • Image: The visual representation or logo of the topic.
  • Page Views: The total number of page views for the topic.

# Translations

This section becomes active after adding the topic, allowing you to translate any topic you desire or delete the translations you've added. This section utilizes the Google Translate infrastructure. Excessive requests may temporarily disable this feature. If you encounter such an issue, please retry after some time.

# AI Features

  • Generate New Content via OpenAI: You can instruct OpenAI to write content on a specific topic. For example: "Write an article about Mobile Application security tips"
  • Regenerate via OpenAI: This section becomes active after adding the topic, allowing you to recreate the content using the OpenAI API.

# SEO Settings

  • Custom Meta Title: Create a custom title for SEO metadata related to the topic.
  • Custom Meta Description: Specify a custom description for SEO metadata.
  • Custom H1 Title: Define a custom H1 title for SEO optimization.

This section is designed for customizing specific SEO elements, including meta title, meta description, and H1 tags. You can use the following supported shortcodes to dynamically generate these elements:

  • %topic_title% Inserts the title of the topic.
  • %site_title% Represents the title of the website.
  • %sep% Adds a separator between elements.
  • %year% Provides the current year.
  • %month% Indicates the current month in numeric format.
  • %day% Shows the current day of the month.
  • %month_text% Presents the name of the current month.
  • %day_text% Displays the name of the current day of the week.

# Add Items to the Topic

After adding or updating a topic, you can view the topic items under this topic by clicking on the 'Click here to view Apps under this topic' link. This allows you to access and review the topic items associated with the topic.

Once you've clicked the link, you can search for the app you want to add using the search box. To add more items to the list, click on 'Add More' After making the desired changes, click the 'Submit' button to save your modifications."
